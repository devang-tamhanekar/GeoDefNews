## How to run (XAMPP on Windows)
1) Install XAMPP from apachefriends.org and start **Apache**.
2) Copy this folder into `C:\xampp\htdocs\` and rename to `geodef-news` (optional).
3) Visit: `http://localhost/geodef-news/` in your browser.
4) You can test PHP using `http://localhost/geodef-news/info.php` (delete later).
